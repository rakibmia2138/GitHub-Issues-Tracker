1. Difference between var, let, and const.

  var → function scoped, can be re-declared and updated <br>
  let → block scoped, can be updated but not re-declared <br>
  const → block scoped, cannot be updated or re-declared 
  
2. What is the spread operator (...)?

  It is used to copy elements of an array or object.
  
3. Difference between map(), filter(), and forEach()

  map() → returns a new array <br>
  filter() → returns filtered array based on condition <br>
  forEach() → only loops, does not return anything 
  
4. What is an arrow function?

  A shorter way to write a function
  
5. What are template literals?

  Strings written using backticks (` `) that allow variables inside
